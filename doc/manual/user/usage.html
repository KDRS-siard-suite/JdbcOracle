JdbcOracle - Users' Manual - Usage

Using the Binaries of the Project

This JDBC driver for DB/2 is to be used like any other JDBC driver. It is recommended, that the static method Db2Driver.register() is used to register it with the DriverManager.

The support for executing SQL is based on a parser for SQL:2008 and is limited to the single-table query functionality needed for SIARD Suite 2.1.

A GUI for JdbcOracle

JdbcOracle is distributed as a collection of JAR files without a graphical user interface. However, the versatile free application SQuirreL SQL Client can be used to make use of it.

UDT-Handling

The Oracle implementation of arrays is called VARRAY and differs from SQL ARRAYs in that they are treated like structured types with a name. The JdbcOracle driver supports the SQL syntax with anonymous arrays whereever possible, implicitly creating or searching VARRAY types with the correct cardinality and element types.
